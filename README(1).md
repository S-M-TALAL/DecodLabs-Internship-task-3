
# NLP Sentiment Analysis Project

## Overview
This project builds a sentiment analysis model using Natural Language Processing (NLP) techniques.

## Features
- Text preprocessing (tokenization, stopword removal, lemmatization)
- TF-IDF vectorization
- Machine learning classification (Naive Bayes / SVM)

## Technologies Used
- Python
- NLTK / SpaCy
- Scikit-learn
- Pandas, NumPy

## Steps
1. Load dataset
2. Preprocess text
3. Convert text to vectors using TF-IDF
4. Train model
5. Evaluate model

## Output
The model predicts whether a given review is positive or negative.
